# HEALTH AI Co-Creation & Innovation Platform

Dockerized full-stack project using:
- React
- Node.js / Express
- PostgreSQL

## Run

`docker compose up --build`

## Services

- Frontend: http://localhost:5173
- Backend: http://localhost:5000/api/health
- PostgreSQL: localhost:5432

## Use of Artificial Intelligence Tools in the Development Process

-   Artificial intelligence tools were actively used throughout the development of the Health AI Co-Creation & Innovation Platform to support different stages of the software engineering process.

-   ChatGPT was used to assist in generating boilerplate code, designing system architecture, and improving overall code structure.

    -   This included backend setup tasks such as Express configuration, route organization, and middleware design.

-   AI tools were utilized for debugging and problem-solving.

    -   They helped identify and resolve issues such as database connection errors, Docker configuration problems, and modüle import failures more efficiently.

-   AI contributed to improving code quality and maintainability.

    -   Suggestions were used for naming conventions, file organization, and best practices in full-stack development using Node.js, React, and PostgreSQL.

-   AI support was also used in documentation and diagram preparation.

    -   Parts of the project report and user guide were drafted with AI assistance.
    -   System design diagrams, including use case and activity diagrams, were initially generated or structured using AI tools and later refined by the team.

-   AI was used during frontend development to assist with component structuring, state management, and routing logic.

-   Despite the extensive use of AI tools, all generated outputs were carefully reviewed, tested, and validated by the development team.

-   Final responsibility for all design decisions, implementation correctness, and system functionality remains with the developers.
